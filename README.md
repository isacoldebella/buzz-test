# Teste de Sobrevivência em Filme de Terror

Uma página web interativa para determinar quem morreria primeiro em um filme de terror baseado em respostas a perguntas.

## Como Usar

1. Abra o arquivo `index.html` em um navegador web (Chrome, Firefox, etc.).
2. Responda todas as 15 perguntas selecionando uma opção por pergunta.
3. Clique em "Calcular Resultado".
4. Veja sua pontuação e o resultado baseado no ranking.

## Estrutura

- `index.html`: Página principal com o quiz.
- `style.css`: Estilos para a aparência da página.
- `script.js`: Lógica JavaScript para calcular a pontuação e mostrar o resultado.

## Ranking

- 🔴 15–25 pts → Morre primeiro
- 🟠 26–40 pts → Morre cedo
- 🟡 41–50 pts → Sobrevive bastante
- 🟢 51–60 pts → Final survivor

## Troubleshooting

- Certifique-se de responder todas as perguntas antes de calcular.
- Se a página não carregar, verifique se todos os arquivos estão na mesma pasta.
